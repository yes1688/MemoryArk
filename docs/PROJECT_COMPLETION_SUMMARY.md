## 🏗️ 技術架構

### 核心技術棧
- **前端框架**: Vue 3 (Composition API)
- **類型系統**: TypeScript
- **樣式框架**: Tailwind CSS + SCSS
- **狀態管理**: Pinia
- **路由**: Vue Router 4
- **構建工具**: Vite
- **代碼檢查**: ESLint + Prettier

### 設計系統
- **設計語言**: Windows 11 Files 風格
- **主題配色**: 教會深藍 + 溫暖金色
- **字體系統**: Segoe UI 字體族
- **圖標系統**: Heroicons 專業圖標庫
- **動畫系統**: CSS 變數 + Vue Transition
