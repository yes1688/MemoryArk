3. **訪問** → http://localhost:7001
4. **管理** → 查看 [用戶指南](../02-user-guide/README.md)

## 💡 快速命令

```bash
# 一鍵安裝
git clone https://github.com/yes1688/MemoryArk.git
cd MemoryArk
./deploy.sh up production

# 常用操作
./deploy.sh status    # 檢查狀態
./deploy.sh logs      # 查看日誌
./deploy.sh restart   # 重啟系統
```
